# IOU Tracker / Splitter App

A simple, fast web app to track shared expenses and calculate who owes whom.

🔗 Live App: https://ledger.comay.ca  

---

## 🚀 Features

- Add transactions in plain English (e.g. "Jake paid $20 for pizza")
- Automatically tracks:
  - Who owes who
  - Total balances
- Clean and simple UI
- Real-time updates
- Supports multiple users

---

## 🛠 Tech Stack

- **Frontend:** Next.js / React
- **Backend:** API routes (Next.js)
- **Parsing:** OpenAI / custom logic
- **Deployment:** Vercel (or similar)
